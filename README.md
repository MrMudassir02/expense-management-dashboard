# 💰 Expense Management Dashboard

A modern and responsive Expense Management Dashboard built with **React**, **Vite**, and **Tailwind CSS**. This application helps users manage their daily expenses with an intuitive interface, real-time statistics, and persistent data storage using Local Storage.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/MrMudassir02/expense-management-dashboard.git
```

Go inside the folder

```bash
cd expense-management-dashboard
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

# 📖 How It Works

### Add Expense

- Enter title
- Enter amount
- Select category
- Click **Add Expense**

---

### Edit Expense

- Click **Edit**
- Update details
- Click **Update Expense**

---

### Delete Expense

Click **Delete** to remove an expense.

---

### Search

Search expenses instantly by title.

---

### Category Filter

Filter expenses based on category.

---

### Statistics Dashboard

Displays

- Total Spending
- Highest Expense
- Total Expense Count
